开始

关于“类”类型：

类定义了行为，不论是自定义类还是标准库定义的类，在定义之初就决定了在此对象上能够使用的所有操作，可以在“标准操作”基础上对某些行为进行重载

点运算符( . ): 只能运用于“类”类型的对象

点运算符( . )与箭头运算符(->)的联系和区别：

  1. 点运算符能够获取类对象的成员，箭头运算符能够获取对象的成员

  2. 点运算直接获取对象的成员，箭头运算符通过指针间接访问对象的成员

  3. 访问对象为指针类型时，需使用箭头运算符；或用点运算此形式：

第二章：变量和基本类型

内置基本数据类型：

根据硬件设计不同、编译器区别、地区或行业统一标准不一致等原因，同一数据类型在不同机器上所占的存储空间有所差异

整型分为：带符号(signed)和不带符号(unsigned)。short、int 、long 、long long都是带符号的，只要在前面加上unsigned就是无符号类型的定义

类型选择的默认准则：

  1. 明确知道数据不可能为负时，选用无符号类型
  2. 常见的整型计算使用 int 定义，长度超过 int 时，选用 long long 类型进行定义
  3. 涉及浮点数运算选择 double 定义

  • 算术表达式中既有 无符号数 又有 int值 时，int值 会转换成 无符号数

  • 把 负数 转换成 无符号数 （类似将一个带符号的值赋给无符号对象）时，此负数存储的位模式不会改变，但是会被解释成一个大正数

字面值常量（具体的数）

字面值常量的形式和值决定了其数据类型：

  • -42是一个有符号的整型
  • 3.1415926是一个无符号(有符号)的浮点数

整型字面值可以用相应的前缀表示其 进制: 0开头为八进制，0x或0X开头表示十六进制

字符字面值和字符串字面值：

字面值类型：可以在编译时求值的类型。常见的基本类型：整型、浮点型、指针类型、空类型void等，均属于字面值类型。string类型、IO库等就不属于字面值类型

  • 字符字面值：用 单引号　括起来的一个字符，只能是单个字符，底层通常表现为对应的ASCII码或Unicode码所对应的值

  • 字符串字面值：用 双引号 括起来的 零个或一个或多个字符 ，不可修改，以空字符 ‘ \0 ‘ 标志结束的常量字符数组，底层表现为一个不可变大小的数组

变量：

变量：一个有具体类型、名字、可供程序操作的 存储空间。定义格式：类型 变量名列表 分号

对象：一块能存储数据，具有某种类型的存储空间

对象的初始化和赋值是有区别的：

  • 初始化：创建变量时赋给其一个初始值
  • 赋值：把对象当前的值擦除，用另一个值来代替

是否初始化对象，由＇类＇决定；如果不进行初始化，那默认的初始化值是什么，也由＇类＇决定

Tip：内置类型的对象，建议进行初始化，避免发生不可靠的事件

复合类型：

复合类型：基于其他类型定义的类型

引用（绑定）

引用不是一个对象，只是给已经存在的对象起了一个别名，并将这个别名和原对象绑定在一起

  • 引用只能绑定在对象上

  • 必须要进行初始化

  • 引用的类型必须要和所绑定的对象类型一致

指针

指针是一个存放地址的对象，可以通过指针对象间接访问其指向的目标(对象)

  • 只进行了定义而未进行初始化的指针是一个危险的存在，不知道此对象指向何处
  • 空指针指向了一个为 nullptr 的值，而不是没有指向任何对象
  • 要访问指针指向的对象，需使用解引用符( * )

const限定符

const修饰的对象，其值不能被改变，必须要进行初始化

const修饰的对象可进行大部分操作，只是不能进行改变对象值的行为

指针和const

const修饰的常量，只能用指向常量类型的指针进行指向

cosnt修饰的常量指针，可以指向一个常量对象，也可以指向一个非常量对象。只是不能通过指针修改其指向对象的值

类型的处理

类型别名

  • 使用 typedef 关键字定义
  • 使用别名声明关键字 using

关键字 auto 和 decltype

auto：当未知对象的类型时，使用 auto 可以使编译器自己分析表达式类型，其声明的对象必须要赋初值，表达式(初值)得到的类型通过 auto 定义其修饰的对象，并以此结果初始化 auto 修饰的对象

decltype：auto 修饰的对象必须要初始化，其行为受到一定限制。decltype可以只自定义其类型，而不进行初始化

第三章：字符串、向量和数组

可变长的字符序列 string

  • 初始化string对象的不同方式
  • 基本操作

关于string的比较：

  1. 大小写敏感
  2. 两个string对象长度不同，且较短的string对象的每个字符都与较长string对象对应位置上的字符相同，则较短string对象小
  3. 两个string对象在某些对应位置上不同，比较结果为：两个对象第一次出现差异的地方，字符大小的比较结果

字符字面值，字符串字面值和string对象

  • 字符字面值：用点引号括起来，只能是单个字符，底层通常表现为对应的ASCII码或Unicode码
  • 字符串字面值：用双引号括起来，零个或一个或多个字符，不可修改，在内存中被分配一个字符数组，以空字符**’\0’**结束
  • string对象：可以进行动态管理，是一个标准库中的类，以空字符**’\0’**结束

头文件 cctype 中的部分函数

cctype中的函数通常用于处理单个字符，例如对string对象某一位上的字符进行处理
