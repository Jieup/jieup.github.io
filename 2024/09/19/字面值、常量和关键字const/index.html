字面值、常量和关键字const

发表于 2024-09-19 | 分类于 Learn ， C++ |
字数统计: 836

C++中的三个比较重要，也是比较容易混淆的基础概念：字面字、常量和const

关键字const

const 是一个比较常见，也是比较重要的关键字。最简单的解释：const修饰的变量，其“值”、也就是内容，不可修改，一经初始化即不可修改。

const与指针

const的引用

和其他引用一样，const的引用也是一个“绑定”

字面值

字面值：直接出现在代码中的值

例如：

  • 整数字面值：例如 42, 0, -7

  • 浮点数字面值：例如 3.14, -0.001

  • 字符字面值：例如 ‘a’, ‘B’, ‘\n’

  • 字符串字面值：例如 “Hello”, “World”

  • 布尔字面值：例如 true, false

字符字面值、字符串字面值和string对象

字符字面值：关键字char ，用单引号括起来的，只能是单个字符，底层通常表现为对应的ASCII码值或Unicode值

字符串字面值：用双引号括起来的，不可修改，可以是零个或一个或多个字符构成，在底层表现为一个字符数组，以空字符”\0”标志结束

string对象：可以进行动态管理，标准库中的类，以空字符”\0”标志结束

常量

常量的定义：不可改变的量

从字面上理解即可，出现后就不能改变的量。又分为两种：

  • 字面值
  • const修饰的变量
